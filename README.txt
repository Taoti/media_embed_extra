# Media Embed Extra
===
Provides additional options for embedded media entities.

## Requirements

- Media

## Installation

- Enable core Media
- Enable Media Media Embed Extra

## Usage

This will provide additional options through the embedded media dialog. Once
an media entity is inserted, edit the embedded media entity and additional
options should show up.

### Configuration

Once installed, you need to ensure the "Embed media" filter is enabled for the
corresponding format (e.g. Full HTML) and if "Limit allowed HTML tags" filter
is enabled, make sure that "<drupal-media>" tag is allowed, as well as the
attributes `data-width` and `data-height`.
